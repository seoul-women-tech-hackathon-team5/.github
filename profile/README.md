## 📌 Introduce
> **OzaOza** : 좋은 세상을 위한 소중한 연결, 착한 가게 활성화 큐레이션 플랫폼   

**착한 소비**, 들어보셨나요?   
세상을 변화시키고 지속가능한 선한 영향력을 행사하며, 자신이 지향하는 가치관에 따라 소비하는 방식입니다.   
'**착한 소비 활성화, 본질에 집중할 수 있는 방법이 없을까?**'라는 생각으로   
OzaOza 서비스를 만들었습니다.
